# 💸 Apportion - Divisão de Gastos Inteligente

O **Apportion** é um aplicativo mobile desenvolvido com **React Native** e **Expo**, focado em simplificar a divisão de despesas entre amigos, viagens e eventos. Com uma interface *Premium Dark*, o app oferece uma experiência fluida para gerenciar quem deve para quem.

---

## ✨ Funcionalidades (Front-End)

- 🎨 **Interface Moderna:** Design focado na paleta Dark (Roxo e Cinza Escuro).
- 🔐 **Autenticação Simples:** Login intuitivo para identificação do usuário.
- 👥 **Gestão de Grupos:** Criação e entrada em grupos com seleção de amigos.
- 👤 **Perfil Personalizável:** Integração com a galeria para foto de perfil e ID único.
- 💬 **Interação:** Lista de amigos com busca em tempo real e interface de chat.
- ⚙️ **Configurações:** Controle de notificações, biometria e limpeza de cache.
- 📱 **Navegação Fluida:** Menu lateral (Drawer) customizado para acesso rápido.

---

## 🛠️ Tecnologias Utilizadas

* [React Native](https://reactnative.dev/)
* [Expo Go](https://expo.dev/)
* [Expo Router](https://docs.expo.dev/router/introduction/) (Navegação baseada em arquivos)
* [Lucide React Native](https://lucide.dev/icons) (Ícones)
* [AsyncStorage](https://react-native-async-storage.github.io/async-storage/) (Persistência de dados local)
* [Expo Image Picker](https://docs.expo.dev/versions/latest/sdk/image-picker/) (Acesso à galeria)
